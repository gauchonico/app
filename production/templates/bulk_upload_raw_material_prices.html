CSV Format Instructions

Required Columns:
  • raw_material_name - Exact name of the raw material (case-insensitive)
  • supplier_name - Exact name of the supplier (case-insensitive)
  • price - Price as a decimal number (e.g., 150.00)
Optional Columns:
  • effective_date - Date in YYYY-MM-DD format (defaults to current date)
  • is_current - Set to 'true' or 'false' (defaults to 'true')
Example CSV:
raw_material_name,supplier_name,price,effective_date,is_current
Sugar,ABC Suppliers,150.00,2024-01-15,true
Flour,XYZ Company,200.50,2024-01-15,true
Salt,ABC Suppliers,25.00,2024-01-15,false
Note: When you set is_current to 'true' for a material-supplier combination, all other existing prices for that combination will automatically be set to 'false'.
